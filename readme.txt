Oxygen Sample Plugin

Un simplu plugin de Oxygen care inverseaza X cu Y la tastare.

Impachetare:
  a) din Eclipse: click dreapta, Run As, Maven Install
  b) din linia de comanda: se ruleaza "mvn clean package"

Instalare:
  se extrage arhiva target\oxygen-sample-plugin-0.0.1-SNAPSHOT-plugin.jar in [Oxygen-Install-Dir]/plugins.
